# Battle Ship **[ 2021 ]**
## Features
 - Saves the amount of guesses used & uses that as the goal of the game. 
 - Auto Generates position of the battle ships. 
 - Board size & amnt of battleships is configurable. 
## GUI
 - Elegant Battleship game running on python turtle graphics. 
## CLI
 - Uses CLI to get information about the game. 
 - Uses numbers and tick markers to show if a battleship is hit / sunk.
